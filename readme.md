# 基于python tornado 应用 **开发中，请勿使用**

## ./app.py

### mac osx
* `$ pip install tornado`
* `$ brew install mongodb`

### windows
* [tornadoweb官网](http://www.tornadoweb.org/)
* [mongodb官网](http://www.mongodb.org/)

### 运行
* 在程序目录下创建`temp`文件夹，然后终端运行`$ ./app.py`
* 提供启动参数`./app.py --help`
* -port 设置端口
* -debug 前端js文件是否输出 debug版本文件
